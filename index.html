The Ins and Outs of NGINX

Abstract: NGINX is a high performance, load balancing, open source web-server that is designed for massive concurrency, low resource usage, and compatibility with modern web architecture.

Key Concepts: Nginx's event-driven workers and reverse proxy features make it ideal for load balancing, caching, TLS offload, and API gateways.

Key Terms:

  • C10K
  • Event loop
  • Worker process
  • Reverse proxy
  • TLS termination
  • Caching
